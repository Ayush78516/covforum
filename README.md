# COV - Council of Valuers Project

This is a full-stack internship project featuring a React frontend and a Node.js/Express backend, integrated with CCAvenue for payments and Redis for OTP/Session management.

## 🚀 Project Structure

- **`/front`**: React.js frontend application.
- **`/Internship-project`**: Node.js/Express backend server.
- **`package.json`**: Root configuration for managing both projects together.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, CSS (Vanilla), html2pdf.zip
- **Backend**: Express.js, MongoDB (Mongoose), Redis, NodeMailer
- **Payment Gateway**: CCAvenue Integration

## 📥 Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-link>
   ```
2. Install all dependencies and build the frontend:
   ```bash
   npm run build
   ```
3. Create a `.env` file in the `/Internship-project` folder with your credentials.

## 🏃 Running Locally

- To start the backend and serve the frontend:
  ```bash
  npm start
  ```
- For development:
  ```bash
  npm run dev
  ```
